# FIX Install Office 2013

Keep Learning, Developing, Sharing and Improving.

**Solusi 1**
1. Uninstall Office di Uninstall Program Control Panel.
2. Cek Office pada C:\Program Files atau C:\Program Files (x86), Delete Microsoft Office.
3. Tombol Windows + R, ketik %programdata% kemudian hapus jika ada folder Microsoft Office.

**Solusi 2**
1. Download File Program di link berikut http://support.microsoft.com/kb/2739501 (Program sudah di cantumkan).
2. Install seperti biasa, **Restart**, dan Happy Working...

**Solusi 3**
1. Tombol Windows + R, ketik %programdata%.
2. Cari folder Microsoft Help ubah Microsoft Help.old

**Solusi 4**
1. Klik tombol Windows, cari Task Scheduler.
2. Pada Tab Microsoft, Cari Office, Kemudian Delete.
